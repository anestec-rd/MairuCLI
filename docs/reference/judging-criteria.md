**MairuCLI Strategy:**

- Is there a clear need for this solution?
- Does it have extensibility or scalability?

## Tie Breaking

If two or more submissions are tied:

1. Compare scores on **Potential Value** (first criterion)
2. If still tied, compare **Implementation**
3. If still tied, compare **Quality and Design**
4. If still tied, judges vote

**Implication:** Potential Value is the most important tie-breaker.
